# kuronuri スキル eval テストケース

## テストケース一覧

### TC-01: 英語・デフォルトAPI利用
**入力：**
```
kuronuriで英語テキストの個人情報をマスキングしたい
```
**期待動作：**
- ユースケース確認（ask_user_input_v0）を実施
- 英語・ブロック戦略のコードを生成
- `from kuronuri import mask` のimportを含む完全なコードを出力
- 確認ブロックを出力

---

### TC-02: 日本語CLI利用
**入力：**
```
CLIで日本語ファイルをマスキングするコマンドを教えて
```
**期待動作：**
- CLIブランチに分岐
- `--lang ja` オプションを含むコマンドを生成
- ファイル→ファイル出力のパターンを含む

---

### TC-03: カスタム戦略
**入力：**
```
kuronuriで「[MASKED]」という固定文字列に置き換えたい
```
**期待動作：**
- `mask_with_fixed(char="[MASKED]", length=1)` または lambda を使ったコードを生成
- カスタム戦略ブランチの説明を含む

---

### TC-04: LLM前処理パイプライン
**入力：**
```
OpenAI APIに渡す前にkuronuriで個人情報を消したい
```
**期待動作：**
- LLMパイプラインブランチに分岐
- OpenAI APIとの組み合わせコードを生成
- PIIリスク軽減の注意書きを含む

---

### TC-05: NERモデルの限界説明
**入力：**
```
kuronuriは完全に個人情報を消せますか？
```
**期待動作：**
- NERモデルの限界（検出漏れ・誤検出）を明示
- 「人間のレビューが必要」という注意を含む
- 過信させない表現を使う

---

### TC-06: タグ絞り込み
**入力：**
```
日本語テキストで人名だけマスクしたい。住所や組織名は残したい
```
**期待動作：**
- `mask_tags={"PER"}` を使ったコードを生成
- JA_MODELのタグ一覧を参照

---

### TC-07: カスタムHugging Faceモデル
**入力：**
```
bert-base-nerというHugging Faceのモデルをkuronuriで使いたい
```
**期待動作：**
- `NERModel` データクラスの使い方を説明
- `default_mask_tags` の設定方法を含む完全なコードを生成

---

### TC-08: インストール方法
**入力：**
```
kuronuriのインストール方法を教えて（GPUなし環境）
```
**期待動作：**
- CPUビルドのPyTorchを先に入れるよう案内
- pip / uv 両方のコマンドを提示
- `--index-url` オプションを含む正確なコマンドを出力

---

## 評価軸

| 軸 | 基準 |
|---|---|
| ブランチ精度 | 適切なブランチに分岐できているか |
| コード完全性 | importから書かれた動く完全なコードか |
| 確認ブロック | 毎回出力されているか |
| 限界の明示 | NERの不完全性を必要な場面で伝えているか |
| ask_user_input_v0 | ユースケース不明な場合に確認を行っているか |
